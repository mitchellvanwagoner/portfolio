Capstone Project

Smart Home Automation System

September 2023 - May 2024

Overview

For my senior capstone project, I designed and implemented a comprehensive smart home automation system that integrates various IoT devices, sensors, and control interfaces to create an intelligent living environment.

Key Features

  • Voice-controlled home automation using custom wake word detection
  • Energy monitoring and optimization algorithms
  • Adaptive lighting based on time of day and occupancy
  • Security system with facial recognition
  • Mobile app for remote control and monitoring

Technologies Used

Raspberry Pi 4 ESP32 MQTT Node-RED React Native Python TensorFlow Lite

Project Gallery

Challenges & Solutions

Challenge: Ensuring reliable communication between 20+ IoT devices
Solution: Implemented MQTT broker with QoS levels and automatic reconnection

Challenge: Reducing false positives in occupancy detection
Solution: Combined PIR sensors with computer vision for improved accuracy

Results & Impact

  • Reduced household energy consumption by 32%
  • Achieved 98.5% accuracy in voice command recognition
  • System response time under 100ms for critical commands
  • Successfully deployed in test home for 3-month trial
